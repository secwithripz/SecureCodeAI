
# 🔐 SecureCode AI

SecureCode AI is a powerful AI-driven web application that detects security vulnerabilities in code. It leverages LLMs like Novita AI and Hugging Face to identify issues such as XSS, SQL Injection, and more.

## 🌟 Features

- 🔎 Scan for vulnerabilities in real-time
- 🤖 Powered by advanced AI models (Hugging Face / Novita)
- 🧠 Detects issues like XSS, SSRF, SQLi, Command Injection, etc.
- 🌐 Intuitive UI with markdown output
- 🚀 Fast, responsive frontend with animated user feedback

## 📂 Repository

- [Main GitHub Repository](https://github.com/secwithripz/SecureCodeAI)

## 🚧 Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **AI APIs**: Hugging Face, Novita.ai
